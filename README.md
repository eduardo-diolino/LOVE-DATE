# 🌹 Date Proposal

A modern, interactive, and romantic web application designed to create a memorable date proposal experience. Built with a scalable architecture using React, TypeScript, Vite, Tailwind CSS, Motion, and Firebase.

## ✨ Features

* Elegant and responsive user interface
* Smooth animations and transitions
* Interactive heart rain effects
* Date proposal form
* Real-time data synchronization with Firebase Firestore
* Hidden admin dashboard (Cupid Panel)
* Inline editing of responses
* Trash and restore system
* Permanent deletion with confirmation
* Optimized favicon support
* Celebration video with graceful fallback background

## 🛠️ Tech Stack

* React 18
* TypeScript
* Vite
* Tailwind CSS
* Motion
* Firebase Firestore

## 📦 Installation

### Prerequisites

* Node.js 18 or later
* npm

### Install dependencies

```bash
npm install
```

### Environment Variables

Create a `.env.local` file in the project root and configure the required environment variables.

Example:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
...
```

## 🚀 Run the Project

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

## 🔍 Preview Production Build

```bash
npm run preview
```
